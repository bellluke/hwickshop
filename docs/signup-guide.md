# 🛒 쿠팡 파트너스 가입 가이드 & 주의점

## 가입 절차

### 1단계: 가입 신청
- https://partners.coupang.com 접속
- 쿠팡 계정으로 로그인 (없으면 쿠팡 회원가입 먼저)
- "파트너스 가입" 클릭

### 2단계: 정보 입력
- **사이트 주소 등록** — 티스토리 블로그 URL + hwick.shop
  - 여러 개 등록 가능
  - ⚠️ 실제로 콘텐츠가 있는 사이트여야 함 (빈 사이트 반려 위험)
- **매체 유형** — "블로그" 선택
- **카테고리** — 주력 분야 선택

### 3단계: 승인 대기
- 보통 **1~3영업일** 내 승인
- 반려 시 사유 안내 → 수정 후 재신청

### 4단계: 정산 정보
- 은행 계좌 등록 (본인 명의)
- 세금 정보 입력 (사업자 or 개인)

---

## ⚠️ 가입 전 준비할 것

### 티스토리 블로그 먼저 세팅
- **가입 전에 글 5~10개는 있어야** 함 (승인 심사용)
- 일반 블로그 글이면 OK — 쿠팡 링크 없어도 됨
- 빈 블로그로 신청하면 반려될 수 있음

### meetdaily.life 자체사이트
- 마찬가지로 콘텐츠 있는 상태에서 등록
- Content Lake 글이 이미 있으면 바로 가능

## 🚨 반드시 지켜야 하는 규칙

### 1. 경제적 이해관계 표시 (필수!)
모든 글 하단에 반드시:
```
이 글에는 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.
```
- **최종승인 반려 1위 사유** = 이 문구 누락
- 글마다 넣어야 함 (사이트 footer에만 넣으면 안 됨)

### 2. 금지 행위 9가지
1. ❌ 본인이 직접 클릭 후 구매 (자기 구매 수수료 X)
2. ❌ "최저가", "할인" 등 가격 보장 표현
3. ❌ 쿠팡이 아닌 곳에서 링크 클릭 유도 (카톡 단톡방 스팸 등)
4. ❌ 자동 리다이렉트로 쿠팡으로 보내기
5. ❌ 팝업/팝언더 광고에 링크 넣기
6. ❌ 성인/불법 콘텐츠에 링크 삽입
7. ❌ 쿠팡 상표를 도메인에 사용 (coupang-deals.com 등)
8. ❌ 쿠팡 가격 정보를 스크래핑해서 표시
9. ❌ 이메일 스팸에 링크 삽입

### 3. 콘텐츠 품질
- **저품질 양산 금지** — 같은 날 수십 개 올리면 플랫폼 패널티
- **실질적 콘텐츠 필수** — 링크만 있는 글 = 반려/정지
- **허위/과장 금지** — "이거 안 사면 손해" 같은 표현 주의

| 카테고리 | 수수료율 |
|---------|---------|

- **라스트 클릭 어트리뷰션** (마지막으로 클릭한 파트너가 수수료 가져감)
- **최소 정산**: 월 10,000원 이상
- **정산일**: N+2개월 15일 (예: 2월 수익 → 4월 15일 지급)

## 링크 생성 방식

승인 후 4가지 방법:

1. **상품링크** — 특정 상품 URL 입력 → 파트너스 링크 생성 (우리 메인)
2. **간편링크** — 쿠팡 내 검색어 기반 자동 링크
3. **다이나믹 배너** — 사이트에 배너 삽입 (meetdaily.life용)
4. **API** — 자동화용 (향후 활용)

### 링크 형태
```
https://link.coupang.com/re/AFFILIATE_CODE?itemId=XXXXXX
```

## 플랫폼 전략 (확정)

| 플랫폼 | 콘텐츠 | 수익 모델 | 쿠팡 링크 |
|--------|--------|----------|----------|
| **hwick.shop** | 쇼핑 추천·비교·체크리스트 | 쿠팡 파트너스 | ✅ 본문 직접 삽입 |
| **티스토리** | 쇼핑 추천 (hwick.shop과 다른 버전) | 쿠팡 파트너스 | ✅ 본문 직접 삽입 |
| **meetdaily.life** | 정보·분석 글 (Content Lake) | 애드센스 | ❌ 넣지 않음 |
| **네이버 블로그** | — | — | ❌ 외부링크 페널티 |

### 왜 이렇게 나누나?
- **애드센스**(페이지 체류) vs **쿠팡 링크**(빨리 이탈) = 목표 충돌
- 한 사이트에 두 개 섞으면 둘 다 효율 떨어짐
- meetdaily.life = 정보로 승부 (애드센스)
- hwick.shop = 쇼핑으로 승부 (쿠팡 파트너스)

### hwick.shop 사이트 구조
```
hwick.shop
├── /school/    ← 신학기 준비물
├── /home/      ← 자취·이사·생활
├── /baby/      ← 육아·출산
├── /health/    ← 건강·운동
└── /gift/      ← 선물 추천
```

### 도메인 정리
| 도메인 | 용도 |
|--------|------|
| **hwick.shop** | 쿠팡 파트너스 쇼핑 추천 ("휙 보고 휙 사는") |
| **hwick.store** | 예비 (리다이렉트 or 향후 확장) |
| **huick.shop/store** | 마이크로 쇼핑몰 솔루션 (별도 프로젝트) |
| **meetdaily.life** | Content Lake 정보 콘텐츠 + 애드센스 |
