## 시리즈 구조

hwick.shop에서 이걸 하나의 시리즈로 묶으면:

```
hwick.shop/start/          ← 스타터킷 시리즈 메인
hwick.shop/start/puppy/    ← 강아지 입양
hwick.shop/start/cat/      ← 고양이 입양
hwick.shop/start/running/  ← 러닝 시작
hwick.shop/start/travel/   ← 여행 준비
...
```

### 왜 "/start/" 인가
- "시작"이라는 공통 맥락 → 시리즈 정체성
- 한 글 읽은 사람이 다른 "시작" 글도 볼 확률 높음
- 내부 링크로 체류시간 증가

---

## 통계
- 총 **34개 주제** (기존 40개에서 중복 4개 제외하면 순증 30개)
- 글당 평균 링크 **12~15개**
- 전체 링크 약 **400~500개** → 클릭 기회 극대화

## 우선순위 (시즌 + 검색량)
1. 🔥 P01 강아지 입양 — 연중 수요, 검색량 높음
2. 🔥 P02 고양이 입양 — 연중 수요, 검색량 높음
3. 🔥 T01 해외여행 짐싸기 — 봄 여행 시즌
4. SP01 러닝 시작 — 봄 운동 시즌
5. SP05 캠핑 입문 — 봄~여름
6. L03 첫 차 용품 — 연중
