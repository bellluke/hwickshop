## 모바일 우선 설계

- **max-width**: 720px (글 본문)
- **카드 그리드**: 모바일 1열, 태블릿 2열, 데스크탑 3열
- **상품 카드**: 모바일에서 가로 풀 너비
- **TOC**: 모바일에서 상단 접히는 아코디언
- **CTA 버튼**: 최소 48px 높이 (터치 타겟)
- **체크리스트**: 터치로 체크 가능

## 핵심 인터랙션
1. **체크리스트 체크** → 체크한 항목 합계 자동 계산
2. **예산 필터** → 절약/표준/넉넉 탭 클릭 시 해당 상품만 표시
3. **목차 하이라이트** → 스크롤 위치에 따라 현재 섹션 표시
4. **공유 버튼** → 카카오톡, URL 복사
