# 🖼️ 쿠팡 파트너스 이미지 전략

## 이미지 유형별 활용

### 1. 상품 이미지 — 쿠팡에서 가져오기
- **쿠팡 파트너스 API** 통해 정식 상품 이미지 URL 사용 (가장 안전)
- 또는 상품 페이지 캡처 (가격 포함 캡처 → 생동감)
- **주의**: 쿠팡 이미지 직링크는 핫링크 방지될 수 있음 → 다운로드 후 자체 호스팅
- 상품당 1~2장이면 충분 (대표 이미지 + 사용 장면)

### 2. 비교표/차트 — 직접 제작 ⭐
Content Lake 스타일로 우리가 만드는 것. **차별화 핵심.**
- **가격 비교 차트**: 상품 5개 가격을 바 차트로
- **스펙 비교표**: 핵심 스펙 3~5개만 시각화
- **비용 계산 그래프**: "1년 기저귀 비용 브랜드별 비교" 등
- **체크리스트 인포그래픽**: 준비물을 카테고리별로 시각 정리
- 색상: 카테고리별 accent (HOME=#8B5CF6, BABY=#F59E0B, HEALTH=#10B981, SCHOOL=#6366F1)

### 3. 썸네일 — 클릭률 좌우
- **공식**: [숫자] + [핵심 키워드] + [이모지/아이콘]
- 예: "입학 준비물 23가지 ✅" / "자취 필수템 TOP 5 💰"
- 텍스트 크고 굵게, 배경은 카테고리 컬러
- 티스토리/네이버 검색 결과에서 눈에 띄어야 함

### 4. 실사용 느낌 이미지
- Unsplash/Pexels에서 상황 사진 (아이 책가방 메고 있는 모습 등)
- "내가 직접 찍은 것 같은" 자연스러운 톤
- 상품 자체보다 **사용 맥락**을 보여주기

## 제작 방식

| 유형 | 도구 | 자동화 가능 |
|------|------|------------|
| 상품 이미지 | 쿠팡 캡처/API | ✅ 스크래핑 |
| 비교 차트 | HTML/CSS → 캡처 | ✅ 템플릿화 |
| 인포그래픽 | HTML/CSS → 캡처 | ✅ 템플릿화 |
| 썸네일 | HTML → 캡처 | ✅ 템플릿화 |
| 상황 사진 | Unsplash API | ✅ 키워드 검색 |

**핵심: 전부 HTML/CSS로 만들고 캡처** → 일관된 브랜드 느낌 + 자동화 가능

## 글 유형별 이미지 구성

### 📋 체크리스트 글 (S01 초등 입학 준비물)
1. 썸네일 (1장)
2. 카테고리별 체크리스트 인포그래픽 (2~3장)
3. 주요 상품 이미지 (3~5장) — 쿠팡 캡처
4. 예산별 총비용 비교 차트 (1장)

### 🏆 TOP N 추천 글 (S04 노트북 추천)
1. 썸네일 (1장)
2. 상품별 이미지 (5장) — 쿠팡 캡처
3. 스펙 비교표 (1장) — 직접 제작
4. 가격대별 추천 정리 (1장) — 인포그래픽

### 💡 문제해결 글
1. 썸네일 (1장)
2. 문제 상황 사진 (1장) — Unsplash
3. 비용/효과 비교 차트 (1~2장) — 직접 제작
4. 추천 상품 이미지 (2~3장) — 쿠팡 캡처
