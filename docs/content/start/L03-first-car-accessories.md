## ❌ 처음에 안 사도 되는 것

- **고급 시트 커버** — 새 차 시트가 제일 깨끗. 오래 타고 나서
- **LED 실내등 교체** — 순정으로 충분
- **윈도우 틴팅 추가** — 출고 시 기본 틴팅 있는 경우 많음. 확인 후
- **접이식 테이블** — 차에서 밥 안 먹으면 불필요
- **차량용 공기청정기** — 에어컨 필터 교체가 더 효과적
