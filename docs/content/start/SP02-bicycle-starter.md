## ❌ 처음에 안 사도 되는 것

- **클릿 슈즈** — 초보가 끼면 정지 시 넘어짐. 3개월 뒤에
- **사이클링 저지 (전문 의류)** — 일반 운동복으로 충분
- **카본 프레임 자전거** — 입문에 200만원은 과투자
- **실내 자전거 트레이너** — 밖에서 먼저 타세요
- **액션캠 (고프로)** — 있으면 멋지지만 필수는 아님
