# 러닝 시작할 때 필요한 것 — 진짜 필수만 (2026)

> 러닝은 돈 안 드는 운동이라고요? 맞아요. 근데 러닝화 하나는 제대로 사야 합니다. 나머지는 있으면 좋은 것.

---

## 러닝, 장비가 중요한가요?

**러닝화 빼고는 전부 선택입니다.** 면 티셔츠에 운동화 신고 나가면 그게 러닝.

하지만 **러닝화만큼은 투자하세요.** 일반 운동화로 뛰면 무릎, 발목, 정강이에 무리가 오고, 2~3주 만에 포기하게 됩니다.

## 초보 러닝 플랜 (첫 4주)

| 주차 | 내용 | 시간 |
|------|------|------|
| 1주 | 걷기 5분 → 가볍게 뛰기 1분 → 걷기 5분 (반복 3세트) | 30분 |
| 2주 | 걷기 3분 → 뛰기 3분 (반복 4세트) | 25분 |
| 3주 | 걷기 2분 → 뛰기 5분 (반복 3세트) | 25분 |
| 4주 | 연속 15~20분 뛰기 시도 | 25분 |

주 3회, 하루 걸러 하루. **속도는 무시.** 완주가 목표.
