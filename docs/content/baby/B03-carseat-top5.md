# 아기 카시트 추천 TOP 5 — 신생아~영아, 안전이 먼저 (2026)

## 카시트, 왜 중요한가요?

- **법적 의무** — 6세 미만 카시트 미착용 시 과태료 6만원
- **사고 시 생존율** — 카시트 착용 시 치사율 70% 감소
- **퇴원 시 필수** — 병원에서 카시트 없으면 퇴원 못 하는 곳도 있음

## 설치 팁

1. **ISOFIX 포인트 확인** — 뒷좌석 시트 틈에 금속 고리 있는지 확인
2. **후방 먼저** — 2세(또는 제품 권장)까지 무조건 후방
3. **조수석 X** — 에어백 위험. 무조건 뒷좌석
4. **흔들림 체크** — 설치 후 양손으로 흔들어서 2.5cm 이상 움직이면 재설치
5. **중고 주의** — 사고 이력 있는 카시트는 절대 X (내부 손상 보이지 않음)
