# 출산준비물 체크리스트 — 진짜 필요한 것만 (2026)

> 출산 준비, 검색하면 100가지가 넘어서 오히려 혼란스럽죠. 산부인과 간호사 + 둘째맘들이 "이건 꼭" / "이건 필요 없었다"로 나눈 리스트입니다.

---

## 핵심 원칙

1. **최소한만 사고, 나머지는 낳고 나서** — 아기마다 다름
2. **선물로 받을 것 감안** — 배냇저고리, 이불, 장난감은 선물로 쏟아짐
3. **시즌 고려** — 여름 출산인데 겨울 내복 미리 살 필요 없음

## ❌ 미리 안 사도 되는 것

- **범보 의자** — 5~6개월 이후. 미리 사면 몇 달 먼지 쌓임
- **이유식 기구** — 6개월 이후. 그때 가서 사도 늦지 않음
- **보행기** — 소아과에서 비추하는 추세
- **고가 장난감** — 신생아는 장난감에 관심 없음
- **예쁜 외출복** — 3개월까지는 거의 안 나감. 집에서 입는 편한 옷이 전부
