## 뭘 써야 할까?

### "아기 피부가 민감하다" → 프리미엄
- 하기스 네이처 메이드 or 팸퍼스 퓨어
- 특히 신생아 때(0~3개월)는 프리미엄 추천
- 피부 트러블 없으면 3개월 뒤 가성비로 전환해도 OK

### "가성비가 중요하다" → 보솜이 리얼코튼
- 면 함유라 피부 자극 적으면서 가격 합리적
- 프리미엄과 최저가의 중간

### "최저가" → 마미포코
- 기본 기능(흡수, 방수)은 다 됨
- 피부 민감하지 않은 아기라면 충분

## 절약 팁

1. **대용량 구매** — 장당 가격 20~30% 저렴
2. **쿠팡 정기배송** — 추가 5~10% 할인
3. **사이즈 넘어갈 때 주의** — 현재 사이즈 대량 구매하면 사이즈 아웃 위험
4. **밤에만 프리미엄** — 낮에는 가성비, 밤(긴 시간)에만 프리미엄 쓰는 맘도 있음
5. **기저귀 파우치** — 외출 시 필요한 만큼만 (아기 가방 무게 줄이기)
