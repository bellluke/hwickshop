## ❌ 자취 초보가 괜히 사는 것들

- **대형 수납장** — 원룸엔 들어갈 데 없음. 행거 + 수납박스 조합이 나음
- **4인용 식기세트** — 혼자 사는데... 1~2개씩 필요할 때 추가
- **커피머신** — 전기포트 + 드립이면 충분. 커피머신은 관리도 귀찮음
- **TV** — 노트북이나 태블릿이면 됨. 넷플릭스 시대
- **다리미** — 스팀 다리미보다 옷걸이 스티머가 자취생한텐 나음
- **정수기** — 브리타 정수 물병이면 충분 (정수기 렌탈은 고정비)

## 자주 묻는 질문

**Q. 가전은 중고로 사도 되나요?**
전자레인지, 선풍기, 전기포트는 중고 OK. 냉장고, 세탁기는 고장 시 수리비가 비싸서 새 것이 낫습니다.

**Q. 원룸에 세탁기가 없어요.**
빨래방 + 소형 건조대가 현실적. 미니 세탁기(3kg)도 있지만, 빨래방이 더 편해요.

**Q. 이케아에서 한꺼번에 사는 게 낫지 않나요?**
배송비 + 조립 시간 생각하면 아이템별로 쿠팡이 더 나은 경우 많습니다. 특히 큰 가구는 이케아가 좋지만, 생활용품은 낱개 배송이 편해요.
