# 🎯 쿠팡 파트너스 전략

> 2026-02-21 확정

## 핵심 원칙

1. **사람처럼 스크래핑** — 3~8초 랜덤 딜레이, 시간당 30~50페이지 상한, 활동시간 09~23시
2. **랭킹 상위만 수집** — 카테고리별 베스트 1~10위, 전체 크롤링 절대 안 함
3. **월 1~2회 갱신** — 시즌 바뀔 때만, 글 쓸 때 해당 카테고리만 확인
4. **콘텐츠가 본질** — 상품 DB가 아니라, 사람들이 읽고 싶은 글을 통해 자연스럽게 쿠팡으로 유입

---

## 차별화 전략: "숫자로 증명하는 쇼핑 가이드"

Content Lake의 강점(숫자 분석, 팩트 기반)을 쿠팡 콘텐츠에도 적용:

- ❌ "이 제품 좋아요~" (감성 리뷰)
- ✅ "에어컨 하루 8시간 = 전기세 2,400원. 서큘레이터 추가하면 30% 절약" (숫자 증명)
- ✅ "자취 첫 달 생활용품 총 비용: 47만원. 항목별 내역 공개" (체크리스트)
- ✅ "10평 원룸 공기청정기 — 소비전력 기준 연간 전기세 비교" (비교 분석)

### 우리만의 톤
- **정보 70% + 상품 30%** — 글 자체가 가치 있어야 함
- **솔직한 단점 언급** — "이건 솔직히 비쌈. 하지만..."
- **숫자 근거** — 감성이 아닌 데이터로 설득
- **상황별 맞춤** — "당신이 OOO라면 → 이거"

## 첫 공략 카테고리 3개

### 1. 🏠 HOME (생활/가전) — 수수료 3~5%
**왜:** Content Lake HOME 클러스터와 시너지, 이사/자취 시즌(2~4월), 상품 다양
**타겟:** 자취생, 신혼부부, 이사 예정자
**키워드 방향:** 상황별 + 문제해결형

### 2. 👶 BABY (육아/출산) — 수수료 5%
**왜:** 수수료 최고 수준, 전환율 높음(부모는 검색 후 바로 구매), 커뮤니티 공유 활발
**타겟:** 예비맘, 초보맘, 둘째 준비
**키워드 방향:** 체크리스트 + 시기별 준비물

### 3. 💪 HEALTH (건강/운동) — 수수료 5%
**왜:** 수수료 5%, 1~3월 다이어트 시즌, 에버그린 수요
**타겟:** 다이어트 시작하는 사람, 홈트족, 건강 관심층
**키워드 방향:** 문제해결 + 비교분석

## 안전장치 (스크래핑)

- HTTP 429 → 즉시 중단, 1시간 쿨다운
- CAPTCHA → 중단, 다음날 재시도
- 403 → 해당 사이트 24시간 스킵
- 연속 실패 3회 → 전체 중단, 수동 점검
- 한 번 긁은 데이터 캐시 → 재방문 안 함
