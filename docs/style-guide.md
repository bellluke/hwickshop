## 이미지 스타일

- 상품 이미지: 흰 배경, 그림자 살짝 (`box-shadow: 0 2px 8px rgba(0,0,0,0.08)`)
- 인포그래픽: 카테고리 액센트 컬러 + 흰 배경
- 썸네일: 카테고리 컬러 배경 + 큰 숫자 + 키워드
- 라운드: 12px (카드), 8px (버튼), 50% (아바타)
