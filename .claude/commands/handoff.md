세션 종료를 대비하여 `.process/handoff.md`에 인수인계 정보를 기록합니다.

아래 항목을 모두 포함:
- **현재 진행 중인 WBS**: 파일 경로
- **마지막 완료 단계**: 구체적 단계명
- **다음 작업**: 다음에 해야 할 것
- **작업 상태**: `계획 완료` / `구현 완료` / `검토 중` 등
- **주의사항**: 진행 중 발견된 이슈, 의존성, 결정사항, 미해결 피드백
- **마지막 갱신**: 타임스탬프

기록 전에 현재 상태를 정확히 파악하기 위해:
1. 현재 WBS 파일 읽기
2. 최근 작업 result 파일 확인
3. 미완료/진행 중 항목 확인

기록 후 handoff.md 내용을 사용자에게 보여줍니다.
