# 3-2 계획: 인터랙티브 체크리스트 컴포넌트 (React)

## 목표
체크 시 합계 자동 계산, localStorage 상태 저장 기능이 있는 체크리스트

## 구현 상세
- React client:load 컴포넌트
- Props: id (localStorage 키), items (name, price, link, category)
- 체크 시: line-through, 합계 자동 계산
- localStorage: 체크 상태 저장/복원 (재방문 시 유지)
- 카테고리 그룹핑: 필수/선택/나중에

## 셀프 검토
- BUILD-PROMPT.md 체크리스트 HTML 구조 반영
- nofollow sponsored 링크 유지
- 가격 포맷: 천단위 쉼표
